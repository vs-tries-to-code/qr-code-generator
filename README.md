# QR Code Generator

A simple web app that generates QR codes from user input. Built using Flask and deployed online.

## Features

* Generate QR codes from any URL or text
* Download generated QR codes
* (Optional) Customize QR colors

## Live Demo

[Click here to try it](https://qr-generator-rgui.onrender.com/)

## Tech Stack

* Python
* Flask
* HTML/CSS
* qrcode library

## Installation (Run Locally)

1. Clone the repository:

```
git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
cd YOUR_REPO_NAME
```

2. Install dependencies:

```
pip install -r requirements.txt
```

3. Run the app:

```
python app.py
```

4. Open in browser:

```
http://127.0.0.1:5000/
```
